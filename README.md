# DINTGND-model: Chaotic Synchronization and Video Encryption

## Introduction
This repository contains the complete MATLAB implementation of the DINTGND model based on Chua's circuit, along with various other chaotic synchronization models such as NTGND, NNRZND, RACZND, ANFOZND, and GND. Furthermore, this project demonstrates the practical application of these chaotic systems in the field of secure video encryption. 

## Updates
* **April 2026**: Fixed a console I/O issue in the ODE solver functions to support extended simulation times without memory overflow. Enhanced the main.m evaluation script by introducing a slope-check mechanism to better distinguish steady states from transient phases during convergence evaluation.

## Features
* **Chaotic Synchronization Analysis:** Simulates and compares the residual errors including Maximum Steady-State Residual Error MSSRE and Average Steady-State Residual Error ASSRE as well as the convergence time CT of multiple neurodynamic models.
* **Video Encryption Application:** Utilizes the chaotic sequences synchronously generated by the master-slave chaotic systems to create dynamic linear noise masks as specified in the application section of the corresponding paper, performing pixel-level XOR encryption and decryption on .avi video files.
* **Comprehensive Security Visualization:** Automatically generates 3D phase portraits, error convergence curves in logarithmic and normal scales, image histograms, and 2D/3D adjacent pixel correlation plots for a full-scale security analysis.

## Requirements
* MATLAB R2021b or newer is recommended
* Image Processing Toolbox required for running image processing functions like imhist and imresize

## File Structure
* `main.m`: The main script for running chaotic synchronization comparison experiments and error evaluation.
* `DINTGND_encryption.m`: The main script for executing the video encryption and decryption process and generating visual security analysis.
* `Chua_*.m`, `P_DINTGND.m`: The core Ordinary Differential Equation ODE function libraries defining different chaotic dynamical systems.
* `generate_chaos_mask.m`: The core function that converts chaotic state values into linear noise masks for encryption.
* `correlation_*.m`: Auxiliary plotting functions used to evaluate and visualize adjacent pixel correlations.
* `test01.avi`: A sample video file used for encryption testing.

## Usage
1. Clone this repository to your local machine or download the ZIP archive.
2. Open MATLAB and change the Current Folder to the project directory.
3. **For Dynamical Synchronization Analysis:** Open and run `main.m` directly.
4. **For Video Encryption and Decryption Testing:** Open and run `DINTGND_encryption.m`. Ensure `test01.avi` is in the same directory as the script. Once the program finishes running, it will automatically pop up visual comparisons and analysis charts of the original, encrypted, and decrypted frames.
